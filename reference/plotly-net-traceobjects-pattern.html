Pattern.init (?BGColor, ?FGColor, ?FGOpacity, ?FillMode, ?Shape, ?MultiShape, ?Size, ?MultiSize, ?Solidity)

Full Usage: Pattern.init (?BGColor, ?FGColor, ?FGOpacity, ?FillMode, ?Shape, ?MultiShape, ?Size, ?MultiSize, ?Solidity)

Parameters:
    ?BGColor : Color - When there is no colorscale sets the color of background pattern fill. Defaults to a `marker.color` background when `fillmode` is "overlay". Otherwise, defaults to a transparent background.
    ?FGColor : Color - When there is no colorscale sets the color of foreground pattern fill. Defaults to a `marker.color` background when `fillmode` is "replace". Otherwise, defaults to dark grey or white to increase contrast with the `bgcolor`.
    ?FGOpacity : float - Sets the opacity of the foreground pattern fill. Defaults to a 0.5 when `fillmode` is "overlay". Otherwise, defaults to 1.
    ?FillMode : PatternFillMode - Determines whether `marker.color` should be used as a default to `bgcolor` or a `fgcolor`.
    ?Shape : PatternShape - Sets the shape of the pattern fill. By default, no pattern is used for filling the area.
    ?MultiShape : seq<PatternShape> - Sets the shape of the pattern fill. By default, no pattern is used for filling the area.
    ?Size : int - Sets the size of unit squares of the pattern fill in pixels, which corresponds to the interval of repetition of the pattern.
    ?MultiSize : seq<int> - Sets the size of unit squares of the pattern fill in pixels, which corresponds to the interval of repetition of the pattern.
    ?Solidity : float - Sets the solidity of the pattern fill. Solidity is roughly the fraction of the area filled by the pattern. Solidity of 0 shows only the background color without pattern and solidty of 1 shows only the foreground color without pattern.

Returns: Pattern
